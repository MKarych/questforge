## 📖 О проекте

**Город Приключений** — платформа для создания, проведения и участия в городских играх и интерактивных мероприятиях.

**Ключевые аудитории:**
- **Игроки** — ищут активный досуг в своём городе
- **Авторы** — создают сценарии и зарабатывают на идеях
- **Организаторы** — проводят игры и зарабатывают на билетах

**Метафора:** "Steam для мероприятий" / "Unreal Engine для городских приключений".

## 🎯 Стратегия

| Этап | Фокус |
| :--- | :--- |
| **1. Городские игры** (MVP) | Автоквесты, пешеходные квесты, GPS-игры |
| **2. Расширение механик** (v1.0) | Квизы, викторины, опросы, раунды |
| **3. Экосистема** (v2.0+) | Музеи, школы, фестивали, туризм, корпоративы |

## 👥 Экосистема

| Роль | Суть |
| :--- | :--- |
| **Игрок** | Проходит игры, платит за билеты |
| **Автор-Организатор** | Создаёт и проводит игры |
| **Чистый Автор** | Пишет сценарии, продаёт на маркетплейсе |
| **Чистый Организатор** | Покупает сценарии, проводит игры |
| **Платформа** | Даёт инструменты, берёт комиссию |

## 🎮 Ключевые фичи

- **Конструктор сценариев** — визуальный редактор как в Figma (React Flow)
- **Игровой процесс** — таймеры, 3 уровня подсказок, очки и штрафы
- **Команды** — создание, приглашение участников, рейтинг
- **Профили** — аватарки, статистика, достижения, рейтинг
- **Роли и модерация** — 5 ролей, заявки на организатора

## 🚀 Быстрый старт

### Предварительные требования
- Node.js (v20+)
- Docker Desktop
- Git

### Установка

```bash
# 1. Клонировать проект
git clone https://github.com/mkarych/questforge.git
cd questforge

# 2. Установить зависимости
npm install

# 3. Настроить окружение
cp .env.example .env

# 4. Запустить Docker
docker-compose -f infrastructure/docker/docker-compose.yml up -d

# 5. Накатить миграции
npx prisma migrate dev --name init

# 6. Запустить бэкенд
cd apps/api && npm run dev

# 7. Запустить фронтенд (в другом терминале)
cd apps/web && npm run dev
```

**После запуска:**
- Бэкенд: http://localhost:3000
- Фронтенд: http://localhost:3001
