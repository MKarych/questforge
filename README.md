## 🌍 Миссия

**Мы создаем экосистему, которая помогает людям придумывать, проводить, находить и монетизировать городские приключения.**

Мы строим не просто "движок для квестов". Мы строим **"Операционную систему для мероприятий" (OS for Events)**, где любой человек может стать автором, организатором или игроком.

---

## 🎯 Стратегия: "От города к миру"

| Этап | Фокус | Почему |
| :--- | :--- | :--- |
| **1. Городские игры** (MVP) | Автоквесты, пешеходные квесты, GPS-игры | Самая "болевая" точка. Encounter устарел, организаторы страдают. |
| **2. Расширение механик** (v1.0) | Квизы, викторины, опросы, раунды | Конструктор легко адаптируется под любые форматы. |
| **3. Экосистема** (v2.0+) | Музеи, школы, фестивали, туризм, корпоративы | Становимся "Steam для мероприятий" — стандартом индустрии. |

## 👥 Экосистема (Пять ролей)

| Роль | Суть | Что делает | Что получает |
| :--- | :--- | :--- | :--- |
| **Игрок** | Конечный потребитель | Проходит игры, платит за билеты | Эмоции, досуг |
| **Автор-Организатор** | **Создает и проводит** | Пишет → проводит → продает (опционально) | Зарабатывает, прокачивает навыки |
| **Чистый Автор** | Креативный специалист | Пишет сценарии, продает на маркетплейсе | Пассивный доход |
| **Чистый Организатор** | Предприниматель | Покупает готовые пакеты, проводит игры | Готовый бизнес |
| **Платформа (Мы)** | Инфраструктура | Дает инструменты, берет комиссию | Комиссии, подписки |

## 📈 Лестница роста (Эволюция пользователя)

Мы **выращиваем** авторов и организаторов из игроков:

```text
1. Игрок
   (Проходит 2-3 игры, влюбляется в формат)
        ↓
2. Автор-Организатор (Новичок)
   (Открывает конструктор, собирает простую игру для 5 друзей)
        ↓
3. Автор-Организатор (Профи)
   (Проводит платные игры, зарабатывает первые деньги)
        ↓
   ┌──────┴──────┐
   ↓             ↓
4a. Чистый Автор    4b. Чистый Организатор
   (Продает сценарии)   (Масштабирует бизнес)
```

## 🧠 Runtime Model (Ключевое разделение)

Платформа работает по принципу жесткого разделения слоев:

```
┌─────────────────────────────────────────────────────────────┐
│                      PRODUCT LAYER                         │
│  (стратегия, роли, UX, экономика)                          │
└─────────────────────────────┬───────────────────────────────┘
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
┌───────▼──────────┐ ┌────────▼─────────┐ ┌────────▼─────────┐
│     BUILDER      │ │   MARKETPLACE    │ │   DASHBOARD      │
│  (конструктор)   │ │  (продажа игр)   │ │  (аналитика)     │
└───────┬──────────┘ └────────┬─────────┘ └────────┬─────────┘
        │                     │                     │
        └─────────────────────┼─────────────────────┘
                              │
                    ┌─────────▼──────────┐
                    │   GAME ENGINE      │ ← САМОЕ ВАЖНОЕ
                    │ (FSM + Runtime)    │
                    └─────────┬──────────┘
                              │
                    ┌─────────▼──────────┐
                    │ REALTIME SERVER    │
                    │ (WebSocket/Rooms)  │
                    └─────────────────────┘
```

**Правило:** Никакая часть системы (фронтенд, сокеты, база данных) не должна содержать игровую логику. Только Engine.

## 📦 Что мы продаем?

Мы продаем **Game Package (Пакет игры)** — готовый бизнес-комплект для проведения мероприятия.

**В состав входит:**
- 📝 Сценарий (вопросы, загадки, сюжет)
- 🖼 Медиа (фото, видео, музыка, презентации)
- ⏱ Настройки (таймеры, правила, подсчет очков)
- 📋 Инструкция организатора (шаблоны рекламы, правила для игроков)
- 🎨 Оформление (постер, баннеры для соцсетей)

**Типы лицензий:**
- "Одно проведение" (дешево, попробовать формат)
- "Многоразовая" (для одного города)
- "Коммерческая" (для бизнеса)
- "White Label" (полное брендирование)

## 🚀 Быстрый старт (для разработчиков)

```bash
# Клонировать репозиторий
git clone https://github.com/mkarych/questforge.git
cd questforge

# Установить зависимости
npm install

# Запустить инфраструктуру (PostgreSQL, Redis, MinIO)
npm run docker:up

# Запустить API (NestJS)
npm run dev:api

# Запустить веб-панель (Next.js)
npm run dev:web
```
