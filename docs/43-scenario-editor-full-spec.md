# Scenario Editor: Полная дорожная карта конструктора

> **Дата:** 23.06.2026  
> **Статус:** Утвержден  
> **Цель:** Сделать конструктор сценариев мощным, как Unreal Engine Blueprint, но простым, как Figma.

---

## 1. Архитектура конструктора

Конструктор строится на трёх слоях:
┌─────────────────────────────────────────────────────────────────┐
│ СЛОЙ 1: ВИЗУАЛЬНЫЙ РЕДАКТОР (React Flow) │
│ - Холст с блоками и соединениями │
│ - Drag-and-drop, зум, панорамирование │
│ - Выделение, подсветка, анимация │
└─────────────────────────────┬───────────────────────────────────┘
│
┌─────────────────────────────▼───────────────────────────────────┐
│ СЛОЙ 2: ЛОГИКА И СОСТОЯНИЕ (Zustand / Context) │
│ - История (Undo/Redo) │
│ - Буфер обмена (копировать/вставить) │
│ - Состояние сценария (сохранён, изменён, опубликован) │
│ - Переменные и глобальные настройки │
└─────────────────────────────┬───────────────────────────────────┘
│
┌─────────────────────────────▼───────────────────────────────────┐
│ СЛОЙ 3: БЭКЕНД И ДАННЫЕ (NestJS + Prisma) │
│ - Сохранение и загрузка сценариев │
│ - Валидация, публикация, версионирование │
│ - Интеграция с игровым движком (Engine) │
└─────────────────────────────────────────────────────────────────┘
