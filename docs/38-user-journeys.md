# Сквозные сценарии пользователя

> Последнее обновление: 22.06.2026

---

## 1. Организатор: полный цикл игры

### 1.1. Регистрация и вход
1. Перейти на страницу регистрации `/auth/register`
2. Заполнить поля (email, пароль, имя)
3. Нажать "Зарегистрироваться"
4. Подтвердить, что произошёл редирект на `/organizer/dashboard`
5. Выйти и зайти заново через `/auth/login`

### 1.2. Создание сценария
1. Перейти в `/organizer/scenarios/create`
2. Добавить блоки (минимум: START → TEXT → FINISH)
3. Настроить блоки (вопрос, ответ)
4. Сохранить сценарий
5. Проверить, что сценарий появился в списке `/organizer/scenarios`

### 1.3. Публикация сценария
1. Нажать "Опубликовать" на сценарии
2. Проверить, что статус изменился на `PUBLISHED`
3. Проверить, что сценарий остаётся в списке

### 1.4. Редактирование сценария
1. Нажать "Редактировать" на сценарии
2. Изменить текст задания
3. Сохранить
4. Проверить, что изменения сохранились

### 1.5. Создание игры
1. Перейти в `/organizer/games/create`
2. Заполнить поля (название, описание, город, дата, цена)
3. Выбрать сценарий из списка
4. Нажать "Создать"
5. Проверить редирект на страницу игры `/organizer/games/[id]`

### 1.6. Публикация игры
1. На странице игры нажать "Опубликовать"
2. Проверить, что статус игры изменился на `PUBLISHED`
3. Проверить, что игра появилась в публичном каталоге `/games`

### 1.7. Получение ссылки на игру
1. На странице игры скопировать `shareLink`
2. Открыть ссылку в новом окне (или инкогнито)
3. Проверить, что открывается страница игры для игроков

## 2. Команда: регистрация на игру

### 2.1. Создание команды
1. Перейти в `/teams/create`
2. Ввести название команды
3. Нажать "Создать"
4. Проверить редирект на страницу команды `/teams/[id]`

### 2.2. Приглашение игроков
1. На странице команды нажать "Пригласить"
2. Ввести email или ник игрока
3. Отправить приглашение
4. Проверить, что приглашение отправлено (статус "pending")

### 2.3. Регистрация команды на игру
1. Найти игру в каталоге
2. Нажать "Зарегистрировать команду"
3. Выбрать команду из списка
4. Выбрать игроков для участия
5. Подтвердить регистрацию
6. Проверить, что команда появилась в списке участников игры

## 3. Игрок: прохождение игры

### 3.1. Вход в игру (LOBBY)
1. Перейти по ссылке на игру (от организатора или из каталога)
2. Выбрать команду (или ввести название)
3. Подтвердить вход
4. Проверить, что отображается LOBBY с командами

### 3.2. Игровой процесс
1. Дождаться старта игры
2. Просмотреть текущее задание
3. Отправить ответ
4. Получить фидбек (правильно/неправильно)
5. Перейти к следующему заданию
6. Проверить, что таймер и подсказки работают

### 3.3. Финиш
1. Пройти все задания
2. Проверить, что открылся экран финиша
3. Проверить, что отображается:
   - Итоговый счёт
   - Количество правильных/неправильных ответов
   - Время прохождения
   - Место в рейтинге (если есть другие команды)

## 4. Админ: модерация

### 4.1. Создание админа
1. Зайти в Prisma Studio (`npx prisma studio`)
2. Найти таблицу `User`
3. Установить роль `ADMIN` нужному пользователю

### 4.2. Модерация заявок организаторов
1. Войти под админом
2. Перейти в `/organizer/admin/applications`
3. Просмотреть список заявок
4. Одобрить или отклонить заявку
5. Проверить, что статус заявки изменился

### 4.3. Модерация игр
1. Перейти в `/admin/games/pending`
2. Просмотреть список игр на модерации
3. Одобрить или отклонить игру
4. Проверить, что статус игры изменился

## 5. Дополнительные сценарии

### 5.1. Редактирование профиля
1. Перейти в `/profile/edit`
2. Изменить аватар, имя, город, контакты
3. Сохранить
4. Проверить, что изменения отобразились на публичной странице `/profile/[id]`

1. Перейти в `/teams`
2. Проверить, что список команд загружается
3. Проверить, что работает пагинация (если есть)
