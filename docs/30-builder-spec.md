### 3.1. Работа с узлами

| Действие | Описание | Горячая клавиша |
| :--- | :--- | :--- |
| Создать узел | Перетащить блок из палитры на холст | Drag & Drop |
| Удалить узел | Выбрать узел → Delete | Delete / Backspace |
| Копировать узел | Ctrl+C → Ctrl+V | Ctrl+C / Ctrl+V |
| Выбрать узел | Клик по узлу | Клик |
| Настроить узел | Клик по узлу → панель настроек | Двойной клик |
| Переместить узел | Перетащить мышкой | Drag & Drop |

| Действие | Описание | Горячая клавиша |
| :--- | :--- | :--- |

| Удалить соединение | Клик → Delete | Delete / Backspace |
| Настроить соединение | Клик по соединению | Клик |

### 3.3. Работа со сценарием

| Действие | Описание | Горячая клавиша |
| :--- | :--- | :--- |

## 5. Валидация в реальном времени

```typescript
class BuilderValidator {
  validate(scenario: Scenario): ValidationResult {
    const errors: ValidationError[] = [];

    // 1. Проверка: есть ли START
    if (!scenario.nodes.some(n => n.type === 'START')) {
      errors.push({
        code: 'ERR_NO_START',
        message: 'Сценарий должен иметь узел START',
        severity: 'error'
      });
    }

    // 2. Проверка: только один START
    const startNodes = scenario.nodes.filter(n => n.type === 'START');
    if (startNodes.length > 1) {
      errors.push({
        code: 'ERR_MULTIPLE_START',
        message: 'Сценарий не может иметь более одного узла START',
        severity: 'error'
      });
    }

    // 3. Проверка: все узлы достижимы
    const reachable = this.getReachableNodes(scenario);
    const orphanNodes = scenario.nodes.filter(n => !reachable.has(n.id));
    for (const node of orphanNodes) {
      errors.push({
        code: 'ERR_ORPHAN_NODE',
        nodeId: node.id,
        message: `Узел "${node.id}" недостижим из START`,
        severity: 'warning'
      });
    }

    // 4. Проверка: нет бесконечных циклов
    if (this.hasCycle(scenario)) {
      errors.push({
        code: 'ERR_INFINITE_LOOP',
        message: 'Сценарий содержит бесконечный цикл',
        severity: 'error'
      });
    }

    return {
      valid: errors.every(e => e.severity !== 'error'),
      errors
    };
  }
}
```
