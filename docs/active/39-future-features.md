## 1. Типы игр (форматы)

Платформа должна позволять создавать любые игры, которые существуют на рынке, и добавлять новые.

### 1.1. Что уже можно сделать (с текущим конструктором)

| Формат | Описание | Статус |
| :--- | :--- | :--- |
| **Классический квест** | Задания, коды, фото, GPS, QR | ✅ Готово |
| **GPS-маршрут** | Посещение точек по координатам | ✅ Готово |
| **Фото-задания** | Загрузка фото в качестве ответа | ✅ Готово |
| **Викторина** | Вопросы с вариантами ответов | ✅ Готово |

### 1.2. Что нужно добавить (v1.0+)

| Формат | Что добавить | Приоритет |
| :--- | :--- | :--- |
| **Интеллектуальная игра** | Блоки для коллективного решения задач, головоломки, ребусы | 🔴 Высокий |
| **Поиск тайников** | GPS-координаты + описание места (геокэшинг) | 🟡 Средний |
| **Творческий конкурс** | Голосование за фото, видео, тексты | 🟡 Средний |

## 5. План развития (v1.0 → v2.0 → v3.0)

### v1.0 (ближайшее время) — основа

1. ✅ Лобби (LOBBY) — ожидание игроков перед стартом
2. ✅ Рейтинг команд — соревновательный дух
3. ✅ Рейтинг авторов — мотивация для авторов
4. ✅ Платные игры — монетизация для организаторов
5. ✅ Маркетплейс сценариев — экономика для авторов

### v2.0 (через 3-6 месяцев) — расширение

1. Интеллектуальные игры (головоломки, ребусы)
2. Поиск тайников (геокэшинг)
3. Творческие конкурсы (голосование)
4. AI-генерация сценариев
5. VK / Telegram бот

### v3.0 (через 6-12 месяцев) — экосистема

1. AR-объекты
2. Мультиплеер в реальном времени
3. Глобальная карта игр
4. API для разработчиков
5. События в реальном времени (погода, время суток)
