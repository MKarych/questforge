# Анализ Мозгобойня: Интеллектуальные игры в барах

> **Конкурент:** Мозгобойня (Brain Games)  
> **Дата анализа:** 23.06.2026  
> **Цель:** Понять формат интеллектуальных игр и применить в нашей платформе.

---

## 1. Общая информация

**Мозгобойня** — это сеть интеллектуальных игр, которые проводятся в барах и ресторанах. Игроки собираются командами и отвечают на вопросы ведущего.

**Ключевые факты:**
- Возраст: ~10 лет
- Аудитория: Команды от 2 до 10 человек
- Формат: Живые игры в барах + онлайн
- Модель: Франчайзинг (продажа франшизы)

---

## 2. Игровой процесс

### 2.1. Основная механика

1. Команды собираются в баре или онлайн.
2. Ведущий задаёт вопросы (разных типов).
3. Команды обсуждают и записывают ответы.
4. Проверка ответов, начисление очков.
5. Победитель получает призы.

### 2.2. Типы вопросов

| Тип | Описание | Пример |
| :--- | :--- | :--- |
| **Текст** | Вопрос с текстовым ответом | "Как зовут главного героя?" |
| **Фото** | Вопрос по фото | "Кто на фото?" |
| **Аудио** | Вопрос по музыке | "Что за песня?" |
| **Видео** | Вопрос по видео | "Что произойдёт дальше?" |
| **Логика** | Задача на логику | "Сколько треугольников?" |

---

## 3. Архитектура платформы

### 3.1. Основные сущности
Команда → Игра → Вопрос → Раунд → Ведущий → Организатор → Бар

text

### 3.2. Роли

| Роль | Описание |
| :--- | :--- |
| **Игрок** | Участвует в команде |
| **Команда** | Группа игроков, общий результат |
| **Ведущий** | Проводит игру, задаёт вопросы |
| **Организатор** | Организует игры в барах |

### 3.3. Экономика

| Источник дохода | Описание |
| :--- | :--- |
| **Билеты** | Игроки платят за участие |
| **Франшиза** | Продажа прав на проведение игр |
| **Партнёрство** | Бары оплачивают проведение |

## 4. Сильные стороны

| Сильная сторона | Почему это важно |
| :--- | :--- |
| **Социальный формат** | Игры объединяют людей |
