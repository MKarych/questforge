# Анализ Loquiz: Универсальная платформа для любых игр

> **Конкурент:** Loquiz  
> **Дата анализа:** 23.06.2026  
> **Цель:** Изучить универсальный подход и применить лучшие практики.

---

## 1. Общая информация

**Loquiz** — это универсальная платформа для создания игр, ориентированная на корпоративные мероприятия, туризм и образование.

**Ключевые факты:**
- Возраст: ~5 лет
- Аудитория: Корпорации, музеи, туристы
- Формат: Любые игры (квесты, квизы, тайники)
- Модель: Freemium + корпоративные лицензии

---

## 2. Игровой процесс

### 2.1. Основная механика

1. Автор создаёт игру в конструкторе.
2. Игроки скачивают приложение.
3. Проходят игру (квест, квиз, тайник).
4. Получают результаты.

### 2.2. Типы игр

| Тип | Описание |
| :--- | :--- |
| **Квест** | Последовательные задания (текст, фото, GPS) |
| **Квиз** | Вопросы с вариантами ответов |
| **Тайник** | Поиск по GPS-координатам |
| **Сбор** | Сбор предметов и выполнение задач |
