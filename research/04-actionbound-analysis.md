# Анализ Actionbound: Квесты для образования и туризма

> **Конкурент:** Actionbound  
> **Дата анализа:** 23.06.2026  
> **Цель:** Понять формат образовательных квестов и применить в нашей платформе.

---

## 1. Общая информация

**Actionbound** — это платформа для создания и проведения интерактивных квестов, ориентированных на образование, туризм и корпоративные мероприятия.

**Ключевые факты:**
- Возраст: ~8 лет
- Аудитория: Школы, музеи, туристы, компании
- Формат: Квесты с заданиями на смартфоне
- Модель: Freemium + корпоративные лицензии

---

## 2. Игровой процесс

### 2.1. Основная механика

1. Автор создаёт квест в конструкторе.
2. Участники скачивают приложение.
3. Проходят квест по точкам на карте.
4. Выполняют задания (фото, видео, текст, GPS).
5. Получают результаты и обратную связь.

### 2.2. Типы заданий

| Тип | Описание |
| :--- | :--- |
| **GPS** | Прийти в точку и подтвердить |
| **Фото** | Сделать фото по заданию |
| **Видео** | Записать видео |
| **Текст** | Ответить на вопрос |
| **QR-код** | Сканировать код на объекте |
| **Выбор** | Выбрать правильный вариант |
| **Сбор** | Найти и собрать предметы |
