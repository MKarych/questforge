# Анализ Geocaching: Поиск сокровищ по координатам

> **Конкурент:** Geocaching (geocaching.com)  
> **Дата анализа:** 23.06.2026  
> **Цель:** Понять силу геолокационных игр и применить лучшие практики.

---

## 1. Общая информация

**Geocaching** — это глобальная игра в поиск тайников (сокровищ) по GPS-координатам. Существует с 2000 года, объединяет миллионы игроков по всему миру.

**Ключевые факты:**
- Возраст: ~25 лет
- Аудитория: Миллионы игроков в 190+ странах
- Количество тайников: > 3 миллионов
- Модель: Freemium (бесплатный базовый доступ + платная подписка)

---

## 2. Игровой процесс

### 2.1. Основная механика

1. Игрок получает GPS-координаты тайника.
2. Приезжает в указанное место.
3. Ищет контейнер (тайник).
4. Находит его, записывает в логбук.
5. Отмечает находку на сайте.
6. Получает очки и достижения.

### 2.2. Типы тайников

| Тип | Описание | Сложность |
| :--- | :--- | :--- |
| **Традиционный** | Одиночный тайник по координатам | ⭐ |
| **Мульти-тайник** | Несколько точек, ведущих к финальному тайнику | ⭐⭐ |
| **Виртуальный** | Нет физического тайника, нужно ответить на вопрос | ⭐ |
| **Мобильный** | Тайник перемещается по городу | ⭐⭐⭐ |
| **Земляной** | Нужно выполнить серию заданий | ⭐⭐⭐⭐ |

### 3.1. Основные сущности

| Роль | Описание |

## 4. Сильные стороны

| Сильная сторона | Почему это важно |
| :--- | :--- |
| **Простота** | Очень низкий порог входа |
| **Глобальное сообщество** | Миллионы игроков по всему миру |
| **Оффлайн-игра** | Не требует интернета во время поиска |

| **Мобильное приложение** | Идеально для игры на улице |
