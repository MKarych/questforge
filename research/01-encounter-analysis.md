# Анализ Encounter: Платформа для городских игр

> **Конкурент:** Encounter (en.cx)  
> **Дата анализа:** 23.06.2026  
> **Цель:** Понять сильные и слабые стороны, чтобы построить лучшее решение.

---

## 1. Общая информация

**Encounter** — это международная сеть городских игр, существующая с 2004 года. Это не просто сайт, а целая экосистема, объединяющая тысячи организаторов, авторов и игроков по всему миру.

**Ключевые факты:**
- Возраст: ~20 лет
- Аудитория: СНГ, Европа, США
- Форматы: 9+ типов игр
- Модель: Франчайзинг + платные игры

## 2. Игровые форматы

| Формат | Описание | Ключевая механика |
| :--- | :--- | :--- |
| **Схватка** | Классический квест с поиском кодов | Код, GPS, Задания |
| **Точки** | Ориентирование по точкам на карте | GPS, Посещение точек |

| **Викторина** | Игра с вопросами и вариантами ответов | Вопросы, Выбор ответа |
| **Фотоохота** | Творческая игра с фотографиями | Фото, Творчество, Оценка |
| **Фотоэкстрим** | Экстремальные фото-задания | Фото, Время, Точность |
| **Кэшинг** | Поиск тайников по GPS-координатам | GPS, Тайники, Поиск |
| **Мокрые войны** | Экшн-игра с водяным оружием | PvP, Охота, Коды |
| **Конкурс** | Творческий конкурс с голосованием | Видео, Аудио, Голосование |

## 3. Архитектура платформы (Reverse Engineering)

### 3.1. Основные сущности
Игрок → Команда → Игра → Сценарий → Автор → Организатор → Домен → Город → Чемпионат

text

### 3.2. Роли

| Роль | Описание |
| :--- | :--- |
| **Игрок** | Участвует в играх, получает очки, рейтинг |
| **Команда** | Группа игроков, общий результат |
| **Организатор** | Создаёт и проводит игры |
| **Автор** | Пишет сценарии для игр |
| **Агент** | Неигровой персонаж, выдающий задания |
| **Судья** | Оценивает творческие задания (фото, видео) |
| **Наблюдатель** | Следит за игрой, помогает организатору |
| **Администратор домена** | Управляет региональным сообществом |

### 3.3. Экономика

| Источник дохода | Описание |
| :--- | :--- |
| **Взносы за игры** | Игроки платят за участие |
| **Франшиза** | Организаторы платят за право использовать бренд |
| **Домены** | Плата за региональные домены (.ru, .ua, .en) |
| **Корпоративные игры** | Проведение игр для компаний |

## 7. Наша стратегия против Encounter

> **"Encounter — это не конкурент. Это набор требований к нашей системе. Любая их механика должна собираться на нашей платформе без кода."**

**Что мы делаем:**
1. **Универсальный движок** — все их форматы становятся просто шаблонами.
2. **Визуальный конструктор** — любой может создать игру без программирования.
3. **AI-инструменты** — генерация, проверка, помощь.
4. **Маркетплейс** — авторы зарабатывают на своих сценариях.
5. **Современный UX** — тёмная тема, адаптив, скорость.
