# Итоговая продуктовая стратегия: Как мы завоюем рынок

> **Дата:** 23.06.2026  
> **Цель:** Сформулировать единую стратегию на основе анализа конкурентов.

---

## 1. Наше позиционирование

> **"Мы — универсальная платформа для любых игр и мероприятий. Encounter, Geocaching, квизы, корпоративные тимбилдинги — это просто шаблоны на нашей платформе."**

**Ключевое сообщение:**  
*"Создай любую игру без программирования. Проведи её где угодно. Зарабатывай на своих идеях."*

---

## 2. Целевая аудитория (сегменты)

| Сегмент | Кто это | Что им нужно | Как мы привлекаем |
| :--- | :--- | :--- | :--- |
| **Организаторы** | Люди, которые хотят проводить игры и зарабатывать | Простой инструмент, готовые сценарии, приём платежей | Бесплатный тариф, маркетплейс, промо-инструменты |
| **Авторы** | Креативные люди, которые пишут сценарии | Платформа для публикации и продажи | Маркетплейс, роялти 70-90%, защита контента |
| **Игроки** | Люди, ищущие активный досуг | Красивый интерфейс, интересные игры, команды | Каталог игр, отзывы, рейтинги, соцсети |
| **Бизнес** | Компании для тимбилдингов и корпоративов | Готовые решения, простота организации | Корпоративные лицензии, White Label |
| **Образование** | Школы, музеи, университеты | Образовательные квесты, простота использования | Образовательные тарифы, гранты |

| Канал | Для кого | Стратегия |

## 7. Риски и их митигация

| Риск | Вероятность | Решение |
| :--- | :--- | :--- |
| **Организаторы не переходят с Encounter** | Средняя | Дать бесплатный импорт игр, сделать супер-простой онбординг |
| **Авторы не приходят** | Средняя | Дать 90% роялти, сделать конкурсы |
| **Игроки не находят игры** | Высокая | Сделать крутой каталог, реклама в городах |
| **Не хватает контента** | Средняя | Сами создать 20+ демо-игр |
| **Сложность продукта** | Средняя | Сделать подробные гайды, видео-уроки |

## 9. Итог

**Мы строим универсальную платформу для любых игр и мероприятий.**

**Наши главные козыри:**
1. **Универсальность** — один движок для всех форматов.
2. **Визуальный конструктор** — любой может создать игру без кода.
3. **AI** — генерация, проверка, NPC.
4. **Маркетплейс** — авторы зарабатывают, организаторы покупают.
5. **Современный UX** — тёмная тема, адаптив, скорость.

**Наша цель:**
> **"Encounter — это всего лишь один из шаблонов, который можно собрать на нашей платформе за вечер."**
