# Анализ ClueKeeper: Детективные квесты и сюжетные игры

> **Конкурент:** ClueKeeper  
> **Дата анализа:** 23.06.2026  
> **Цель:** Понять формат детективных квестов и применить в нашей платформе.

---

## 1. Общая информация

**ClueKeeper** — это платформа для создания детективных квестов, где игроки распутывают загадки и следуют по сюжетной линии. Особенно популярна в США.

**Ключевые факты:**
- Возраст: ~8 лет
- Аудитория: Любители загадок, семьи, компании
- Формат: Детективные квесты с элементами RPG
- Модель: Покупка отдельных квестов

---

## 2. Игровой процесс

### 2.1. Основная механика

1. Игрок покупает квест (скачивает приложение).
2. Следует по сюжету, решая загадки.
3. Получает подсказки и улики.
4. Распутывает дело и находит ответ.
5. Проходит игру за 1-3 часа.

### 2.2. Типы механик

| Механика | Описание |
| :--- | :--- |
| **Диалоги** | Общение с персонажами |
| **Инвентарь** | Сбор предметов для решения |
| **Локации** | Перемещение между местами |
| **Головоломки** | Решение логических задач |
| **Выбор** | Влияние на сюжет (ветвления) |
| **Подсказки** | Система подсказок (3 уровня) |
| **Финал** | Несколько концовок |

---

## 3. Архитектура платформы

### 3.1. Основные сущности
Автор → Квест → Сцена → Диалог → Персонаж → Предмет → Улика → Игрок

text

### 3.2. Роли

| Роль | Описание |
| :--- | :--- |
| **Игрок** | Проходит квест |
| **Автор** | Создаёт квест |
| **Персонаж (NPC)** | Взаимодействует с игроком |

### 3.3. Экономика

| Источник дохода | Описание |
| :--- | :--- |
| **Продажа квестов** | Игроки покупают отдельные квесты |
| **Подписка** | Доступ к библиотеке квестов |

---

## 4. Сильные стороны

| Сильная сторона | Почему это важно |
| :--- | :--- |
| **Глубокий сюжет** | Игроки погружаются в историю |
| **Ветвления** | Решения игрока влияют на финал |
| **Инвентарь** | Собираем предметы для решения |
| **NPC** | Живые персонажи с диалогами |
| **Разные концовки** | Переиграть несколько раз |

## 5. Слабые стороны

| Слабая сторона | Что мы предложим |
