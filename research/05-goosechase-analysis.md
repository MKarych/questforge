# Анализ GooseChase: Корпоративные игры и тимбилдинги

> **Конкурент:** GooseChase  
> **Дата анализа:** 23.06.2026  
> **Цель:** Понять формат корпоративных игр и применить в нашей платформе.

---

## 1. Общая информация

**GooseChase** — это платформа для создания и проведения корпоративных игр, тимбилдингов и мероприятий. Игроки выполняют задания (миссии) на смартфонах, зарабатывая очки.

**Ключевые факты:**
- Возраст: ~10 лет
- Аудитория: Корпорации, школы, университеты
- Формат: Миссии с фото и видео
- Модель: Подписка (есть бесплатный тариф)

---

## 2. Игровой процесс

### 2.1. Основная механика

1. Организатор создаёт игру с миссиями.
2. Участники делятся на команды.
3. Каждая команда получает задания на телефон.
4. Выполняет миссии (фото, видео, тексты, GPS).
5. Набирает очки и поднимается в рейтинге.

### 2.2. Типы миссий

| Тип | Описание | Пример |
| :--- | :--- | :--- |
| **Фото** | Сделать фото по заданию | "Сфотографируйте всю команду" |
| **Видео** | Записать видео | "Снимите танец всей командой" |
| **Текст** | Ответить на вопрос | "Назовите девиз команды" |
| **GPS** | Посетить точку | "Придите к памятнику" |
| **Сбор** | Собрать предметы | "Найдите предметы красного цвета" |
| **Голосование** | Оценить других | "Проголосуйте за лучшую команду" |

## 3. Архитектура платформы

### 3.1. Основные сущности
Организатор → Игра → Миссия → Команда → Участник → Результат → Рейтинг

text

### 3.2. Роли

| Роль | Описание |
| :--- | :--- |
| **Организатор** | Создаёт игру |
| **Команда** | Группа участников |
| **Участник** | Выполняет миссии |
| **Модератор** | Проверяет выполнение миссий |

### 3.3. Экономика

| Источник дохода | Описание |
| :--- | :--- |
| **Подписка PRO** | Для организаторов (больше участников, расширенные функции) |
| **Корпоративные лицензии** | Для крупных компаний |
| **Платные мероприятия** | Организатор проводит игру за деньги |

| Сильная сторона | Почему это важно |

| **Корпоративный фокус** | Огромный B2B-рынок |

| **Мобильное приложение** | Просто использовать на мероприятии |
